> C로 작성된 유닉스 프로그램은 함수 main에서 부터 시작한다.

# C - Main 함수의 원형
```c
Int main (int argc,char *argv[])

      // argc : 프로그램 인수의 개수

      // argv : 인수 자체를 표현하는 문자열의 배열
```
> Int main (int argc,char \*argv[])
      argc : 프로그램 인수의 개수
      argv : 인수 자체를 표현하는 문자열의 배열
